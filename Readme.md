# AI File System Assistant

This project implements a file system assistant that can read resumes, search keywords, and generate summaries.

## Features

- Read TXT, PDF, and DOCX resume files
- List files in a directory
- Search for keywords in resumes
- Create summary files
- Tool-based assistant architecture

## Setup

Install dependencies:

pip install -r requirements.txt

## Run Program

python llm_file_assistant.py

## Example Queries

Read all resumes

Find resumes mentioning python

Create summary resume_java.txt

## Project Structure

fs_tools.py - File system tools

llm_file_assistant.py - Assistant logic

resumes/ - Sample resume files

summaries/ - Generated summaries
